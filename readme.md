# Slava Savelyev

I build **reproducible simulations** and **privacy-aware civic/volunteer tools**.
I care about clear assumptions, measurable trade-offs, and small systems that are easy to run and audit.

## If you only have 5 minutes

Start here:

- **EconPolicySim** — agent-based toy economy for tax/UBI tradeoffs (metrics + sweeps).  
  → read the model, run 1 command, reproduce plots.
- **OrbitLab** — orbital mechanics / N-body sandbox with integrator benchmarks (energy drift).  
  → compare Euler vs RK4 vs symplectic methods and see stability differences.
- **RescueOps** — privacy-aware volunteer dispatch prototype (FastAPI + Leaflet).  
  → case intake → explainable urgency scoring → route planning without exposing addresses.
- **Lux Punch** — real-time punch analyzer (MediaPipe Pose + OpenCV + PyQt).  
  → kinematics + “force proxy” with calibration and a GUI.

## What I’m exploring
- Numerical methods & stability (energy drift, symplectic integration)
- Agent-based models and “honest metrics” (what you can / cannot claim)
- Privacy-by-design (reduce location leakage while keeping utility)
- Computer vision pipelines (real-time constraints, calibration, measurement error)

## How I work (portfolio signal)
- **Reproducibility first**: one-command runs, pinned dependencies, documented assumptions.
- **Testable core**: keep algorithms in pure functions; test edge cases and invariants.
- **Evidence over vibes**: plots, benchmarks, and written limitations.
